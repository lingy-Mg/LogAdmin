# 任务 01: 项目初始化和基础结构搭建

## 目标
创建 Monorepo 项目结构，配置 pnpm workspace，搭建基础框架。

## 执行步骤

### 1. 初始化根项目
```bash
# 在 LogAdmin 目录下执行
pnpm init
```

### 2. 创建目录结构
创建以下目录：
- `packages/frontend`
- `packages/backend`
- `packages/shared`
- `packages/receivers/http`
- `docs`
- `scripts`
- `docker`

### 3. 配置 pnpm workspace
创建 `pnpm-workspace.yaml` 文件：
```yaml
packages:
  - 'packages/*'
  - 'packages/receivers/*'
```

### 4. 创建根 package.json
配置内容包括：
- 项目基本信息
- 工作区脚本（dev, build, test, lint）
- 共享依赖（TypeScript, ESLint, Prettier）
- engines 配置（Node.js >= 18.x, pnpm >= 8.x）

### 5. 配置 TypeScript
创建 `tsconfig.base.json`：
- 启用严格模式
- 配置路径别名
- 配置编译选项

### 6. 配置 ESLint
创建 `.eslintrc.js`：
- TypeScript 规则
- Vue 3 规则
- Prettier 集成

### 7. 配置 Prettier
创建 `.prettierrc`：
- 缩进、引号、分号等规则
- 忽略文件配置

### 8. 创建 .gitignore
忽略：
- node_modules
- dist
- .env
- logs
- 临时文件

### 9. 创建项目 README.md
包含：
- 项目介绍
- 技术栈
- 目录结构
- 快速开始
- 开发指南

## 验收标准
- [ ] pnpm workspace 配置正确
- [ ] 可以执行 `pnpm install` 成功
- [ ] TypeScript 配置无错误
- [ ] ESLint 和 Prettier 工作正常
- [ ] 目录结构完整

## 预期产出文件
- `package.json`
- `pnpm-workspace.yaml`
- `tsconfig.base.json`
- `.eslintrc.js`
- `.prettierrc`
- `.gitignore`
- `README.md`
- 完整的目录结构
